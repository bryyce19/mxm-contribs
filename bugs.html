Fixed

Overwrite popup doesn't support languages outside of English
Fixed
The overwrite warning popup now fully supports all languages in Musixmatch Studio. Previously, the popup would not appear when the studio was set to languages other than English.
Reported: June 10, 2025
Fixed: June 15, 2025
Version: 5.0.0
Contributor data not loading on very large tracks
Fixed
Implemented lazy loading for contributor data, resolving issues with tracks containing 50+ contributors. The data now loads progressively, ensuring all contributor information is displayed correctly regardless of the number of contributors.
Reported: June 2, 2025
Fixed: June 15, 2025
Version: 5.0.0
Theme toggle not persisting between sessions
Fixed
Added local storage support for theme preferences. Your dark/light theme choice is now saved and automatically applied when you return to the studio.
Reported: N/A
Fixed: June 15, 2025
Version: 5.0.0
Button not appearing on first song load
Fixed
The contributor button may not appear when first loading a song after activating the script. This is a one-time issue that resolves itself when navigating to another song.
Reported: May 22, 2025
Fixed: May 25, 2025
Version: 2.0.0
Loading state gets stuck
Fixed
The loading state may get stuck if the button is clicked before the contributor data is fully loaded. This has been fixed by implementing proper loading state management.
Reported: June 2, 2025
Fixed: June 9, 2025
Version: 4.0.0
Unable to scroll when dialogs are open
Fixed
When certain dialogs (like the track info dialog) were open, the page became unscrollable and you could not scroll the contributors list. This has been fixed: the contributors panel is now always scrollable, even when a native popup is open.
Reported: May 30, 2025
Fixed: June 19, 2025
