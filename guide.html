Musixmatch Contributor Viewer

A tool for viewing contributor information within Musixmatch Studio

Installation

First, Install Tampermonkey

  1. Visit tampermonkey.net
  2. Click "Download" and install the extension to your browser
  3. Once installed, pin the extension to your browser toolbar

Next, Install the Script

  1. Go to the website: bryyce19.github.io/mxm-contribs
  2. Click the Install Script button
  3. Once installed, you're ready to use the script

Using the Script

To Start

Navigate to any mission or non-mission studio page on musixmatch.com.

Accessing Contributor Info

  1. Click the info icon in the studio
  2. The first time, a new button (people emoji) will appear in the bottom left
  3. Click the button to load contributors
